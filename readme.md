##### CoffeePot
Installation instructions for application:
  * `npm install` @ root
  * `npm start` @ root

Installation instructions for server:
  * `npm install` @ src/srv
  * `npm start` @ src/srv


##### JSHint

  * `apm install linter-jshint`
